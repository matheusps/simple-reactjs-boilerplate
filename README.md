# Simple ReactJS Boilerplate

### Dependencies:
* [ReactJS] - HTML enhanced for web apps!
* [Webpack 4] - Module Bundler
* [Babel] - Javascript compiler
* [Sass] - CSS with superpowers

##### Build
```sh
$ yarn
$ yarn build
```

##### Development
```sh
$ yarn start
```
